# GUI and UHID release 2026-07-20

This release is for unlocked Xiaomi Mi 8 Pro (`equuleus`) devices only. It is a
temporary-boot release: use `fastboot boot`, never `fastboot flash`.

## Source revision

- Commit: `a2ab938fb5861f2f73d65fa2a2334454034d451e`
- Kernel: Linux `5.12.0-sdm845`, built with `CONFIG_UHID=y`
- Userspace: Ubuntu 24.04.4 ARM64 with physical landscape Xfce, Xorg fbdev,
  physical-desktop VNC, NetworkManager GUI, Blueman, and input diagnostics.
